Es básicamente un backup de los scripts que uso para automatizar (y no tener que recordar) comandos que uso muy seguido. Y sobre todo en caso de migrar a otra distro o 
tener que resintalar, no tener que empezar de cero.

En la carpeta de .config/dwmbar están los módulos para mostrar información del sistema, torrents, mpc, fecha, etc.
En .local/scripts están los scripts (dah!) que uso para hacer todo, muchos con interfaz del DMENU:
- Volumen
- Bluetooth
- Minimal Torrent client, transmission
- Screencastear
- Control de webcam
- y un montón de otras porquerías

Si bien la posibilidad de que alguien lea este README.md y que use alguno de los scripts es despreciable quizás pueda ser de ayuda para algún principiante de Gnu/Linux (such as myself!).

Bienvenidas será cualquier crítica que me ayude a entender un poco más de esto.
